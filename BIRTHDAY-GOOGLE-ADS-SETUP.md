## STEP 6: Conversion Tracking

### Set Up Conversions:

1. **Form Submission** (Primary)
   - Action: Mold form completed
   - Value: $300 (average order)
   - Count: One per click

2. **Phone Call** (Secondary)
   - Action: Call from ad
   - Duration: 60+ seconds
   - Value: $200

3. **Click to Text** (Tertiary)
   - Action: SMS link clicked
   - Value: $150

### Tracking Code:
Add to mold-form.html thank-you page:
```html
<script>
gtag('event', 'conversion', {
    'send_to': 'AW-XXXXXXXXX/XXXXXX', // Your conversion ID
    'value': 300.0,
    'currency': 'USD',
    'transaction_id': ''
});
</script>
```
